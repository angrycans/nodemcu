ESP32 7轴伺服运动控制器操作说明

本项目是一款高性能的 7轴伺服控制器，专为赛车模拟器（Sim Racing）和飞行模拟器（Flight Sim）的运动座舱控制而设计。通过与主流运动软件配合，实现高精度的姿态模拟。

1. 整体功能描述

该控制台通过 Web 界面与 ESP32 固件进行实时交互，支持对 7 个伺服轴进行精确的位置控制、机械参数配置以及系统级的状态监控。

软件兼容性

控制器原生支持以下主流模拟运动控制软件：

  • FlyPT Mover
  • SimTools
  • SimHub

主要应用于赛车、飞行游戏模拟器的 2DOF/3DOF/6DOF 及更多自由度的运动平台控制。

2. 功能列表

轴控制与配置 (Axis 1 - Axis 7)

每个轴都拥有独立的控制面板，支持以下功能：

  • 状态显示：实时显示轴的启用状态（已启用）及当前位置百分比。
  • 位置控制：通过滑块或手动输入调整轴的当前位置（单位：mm）。
  • 行程设置 (Stroke)：配置该轴的最大工作范围（单位：mm）。
  • 导程设置 (Lead)：配置螺杆导程或每转移动距离（单位：mm/rev）。
  • 电机反向：通过复选框快速切换电机的运行方向。
  • 归零功能：
    • 归零按钮：触发该轴的自动归零流程。
    • 手动归零：手动将当前位置标定为零点。

全局控制

  • 连接管理：显示当前连接状态（如“已连接”），支持断开或重新连接。
  • 自动归零：全局开关，开启后系统可在特定条件下自动执行归零。
  • 转速设置：提供高、中、低等不同档位的全局运行速度调节。
  • 归零全部：一键控制所有轴同时执行归零操作。
  • 控制器重启：远程触发 ESP32 控制器重启。
  • 固件更新：支持在线检查或上传新的固件版本。

系统监控

  • 控制台输出：实时显示底层串口/网络通信的原始数据（TX/RX），方便开发者进行协议分析与故障排除。
  • 清除日志：一键清空控制台日志窗口。
  • 多语言支持：支持中文、英文等语言切换。

3. 快速上手

  1. 确保 ESP32 已正常运行并连接至网络。
  2. 在浏览器访问控制台地址（如 servo.marslife.top）。
  3. 确认左上角显示为“已连接”。
  4. 根据机械结构配置每个轴的“行程”与“导程”。
  5. 点击“归零全部”初始化轴位置，随后即可通过滑块控制各轴运动。
